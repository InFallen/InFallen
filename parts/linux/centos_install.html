Скачивание дистрибутива

Заходим на официальный сайт по ссылке и нажимаем Download : На открывшейся странице выбираем вкладку CentOS Linux и в зависимости от архитектуры вашего компьютера или сервера версию дистрибутива: Выбрав версию вы попадёте на страницу со списком зеркал, откуда можно скачать дистрибутив: Перейдя по одной из ссылок вы увидите список файлов доступных для скачивания(образы и торрент файлы):

0_README.txt
Файл с описание представленных образов и инструкцией по их использованию.
CentOS-7-x86_64-DVD-2009.iso
Этот образ DVD содержит все пакеты, которые можно установить с помощью установщика. Это рекомендуемый образ для большинства пользователей.
CentOS-7-x86_64-NetInstall-2009.iso
Это образ сетевой установки или восстановления. Установщик спросит, где он должен получить пакеты для установки. Этот образ наиболее полезен, если у вас есть локальное зеркало пакетов CentOS.
CentOS-7-x86_64-Everything-2009.iso
Этот образ содержит полный набор пакетов для CentOS Linux 7. Его можно использовать для установки или заполнения локального зеркала. Для этого образа требуется флешка на 16 ГБ, так как он слишком большой для записи на DVD.
CentOS-7-x86_64-Minimal-2009.iso
Целью этого образа является установка очень простой системы CentOS Linux 7 с минимум пакетов, необходимых для работы системы. Все остальное нужно установить с помощью yum. Набор пакетов, устанавливаемых этим образом, совпадает с установленным при выборе установки под названием «Минимальная» из полного образа DVD.
sha256sum.txt
Хэш суммы файлов с образами.

Для дальнейшей установки, вам нужно создать загрузочную флешку со скачанным образом. Дальнейшее описание установки будет идти с использованием образа CentOS-7-x86_64-Minimal-2009.iso.

Установка

Для установки загрузимся с созданной загрузочной флешки и увидим данный экран: На нем выбираем Test this media & install CentOS 7 для проверки созданной флешки и дальнейшей установки.
После прохождения проверки мы переходим к установке и увидим экран с выбором языка: Здесь нажимаем кнопку Continue и переходим к следующему экрану:
