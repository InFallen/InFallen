Полезные расширения

Универсальные расширения

Данные расширения подойдут для всех пользователей Visual Studio Code.

  • vscode-icons - изменение иконок к файлам и папкам, помогает лучше ориентироваться в файлах.
  • Code Spell Checker - проверка орфографии английского языка.
  • Russian - Code Spell Checker - словарь русского языка для Code Spell Checker.
  • Indent Rainbow - цветовое оформление отступов внутри проекта.
  • Better Comments - расширяет возможности по работе с комментариями.
  • Path Intellisense - оснащает редактор возможностями по автозавершению путей к файлам.
  • Import Cost - отображает в редакторе размер импортируемого пакета в одной строчке с соответствующим методом

Расширения Git

Расширения из этого списка помогут улучшить ваш опыт работы с git.

  • GitLens - помогает визуализировать информацию о истории кода в Git и многое другое.
  • Git Graph - визуальное представление для веток Git репозитория. Проще один раз увидеть.

Расширения для верстки

Данные расширения помогут ускорить работу верстальщика.

  • Auto Rename Tag - автоматическое переименовывание парных тегов "HTML/XML".
  • Auto Close Tag - автоматическое закрытие тегов "HTML/XML".
  • CSS Peek - Разрешает просмотр идентификаторов CSS и строк классов как определений из HTML-файлов в соответствующий CSS-файл. Позволяет просматривать и переходить к определению.

Расширения для Python-разработчиков

Эти расширения помогут любому python-разработчику улучшить свой опыт работы с VSCode

  • Python - данное расширение — мощный источник полезных функций:
    • IntelliSense: Редактируйте код с возможностями автодополнения, навигации по коду, проверки синтаксиса и др.
    • Линтинг (Linting): Дополнительно анализируйте код с помощью Pylint, Flake8 и др.
    • Окружения: Автоматическая активация и переключение между окружениями virtualenv, venv, pipenv, conda и pyenv.
    • Тестирование: Запуск и отладка тестов через Test Explorer с помощью unittest, pytest или nose.
    • Форматирование кода: Оформите код с помощью black, autopep или YAPF
  • Python Preview - расширение, которое включает визуальную отладку в ваш код на Python. Оно превращает процесс отладки кода в интерактивное действие с анимацией и графическими элементами для представления состояния вашего приложения.
  • Jupyter - интеграция Jupyter-notebook в Visual Studio code.
