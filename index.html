Полезные ресурсы

На данном ресурсы содержится информация и ссылки на полезные ресурсы по технологиям, которые я использую в работе.

HTML/CSS

  • https://www.w3.org/html/ - официальная документация HTML.
  • https://validator.w3.org/ - валидатор HTML-разметки.
  • Siteliner — инструмент для seo-анализа сайта. Поиск дублирующегося контента, неработающих ссылок и многое другое.
  • HTML Template Generator - сервис для генерации базовой HTML-разметки.
  • https://www.w3.org/Style/CSS/ - официальная документация CSS.
  • Badge Service — генерация svg-значков в стиле github.
  • PX to EM - удобный конвертер из PX в EM и обратно.
  • Fibonacci - визуальный конструктор flexbox-лэйаута.

JavaScript

  • JS Mozilla - документация по JavaScript от Mozilla.
  • JS Nice - сервис по анализу и приведению в порядок кода на JavaScript, добавление отступов, комментариев.
