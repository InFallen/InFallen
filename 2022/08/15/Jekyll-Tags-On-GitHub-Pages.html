Jekyll Tags на GitHub Pages

[ jekyll  blog  github-pages  ]

15 Aug 2022 - Артем 'InFallen' Нагорный

Github Pages по умолчанию не поддерживает плагин jekyll-tagging для работы с тегами в Jekyll.

Для внедрения тегов можно использовать данное пошаговое руководство. Для демонстрации, данный механизм используется на данном сайте.
